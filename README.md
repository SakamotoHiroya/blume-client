# blume-client

blume-clientはblumeコードの生成、読み取りを行うことのできるWebサービスです

blumeコードは花の形をした任意のデータを埋め込める二次元コードです

## 開発背景

ハックツハッカソンステゴカップ提出作品

ハッカソンテーマ「〇〇の魔法」

私たちのテーマは花畑を出す魔法です
